# 📘 Брендбук Tavrida Lot — v0.1 (draft)

> **Статус:** первая версия для обсуждения · **Дата:** 2026-07-08  
> **Назначение:** единые правила визуала и тона для продукта, маркетинга и документации.  
> **Реализация токенов:** `@tavrida/ui` + Tailwind ([14-frontend](../14-frontend/README.md)).

Это **рабочий черновик**, от которого отталкиваемся. После ревью → design tokens → пилот на 2–3 экранах.

## 1. Суть бренда

### Кто мы

**Tavrida Lot** — платформа сообщества вокруг **находок**: аукционы, форум, услуги. Географический якорь — **Крым**, аудитория шире: коллекционеры, краеведы, искатели, те, кому важна история вещи.

### Позиционирование (одно предложение)

> Место, где находка обретает цену, историю и доверенных людей рядом.

### Не позиционирование

| Мы не… | Мы… |
|--------|-----|
| Маркетплейс «как у всех» | Сообщество с торгами и репутацией |
| Антикварный бутик / музейный каталог | Живой, современный продукт «для своих» |
| Крипто/финтех-аукцион | Честные торги с понятными правилами |
| Туристический сувенирный бренд | Пространство вокруг вещи и её судьбы |

### Обещание бренда

1. **Ясность** — лот, ставка, срок, репутация читаются без инструкции.  
2. **Доверие** — рейтинг, отзывы, лимиты работают на честную игру.  
3. **Присутствие** — realtime-торги ощущаются «сейчас», а не как форма на сайте.  
4. **Уважение к вещи** — фото и описание находки важнее декоративного UI.

## 2. Характер и Tone of Voice

### Архетип (смесь)

**Хранитель места** (Крым, контекст) + **Честный торговец** (ставки, цена) + **Знакомый эксперт** (форум без снобизма).

### 5 черт личности

| Черта | В продукте | В текстах |
|-------|------------|-----------|
| **Собранный** | Чистая иерархия, мало шума | Короткие фразы, без канцелярита |
| **Тёплый** | Человечные empty states | «Ты», без холодного «пользователь» |
| **Точный** | Цифры, таймеры, статусы | Конкретные суммы и сроки |
| **Живой** | Пульс ставки, обновления | Настоящее время, глаголы действия |
| **Земляной** | Материал, свет, фактура | Без пафоса «элиты» и без сленга ради сленга |

### Тон: шкала

```
Формальный ●●●○○ Неформальный
Сухой      ●●○○○ Живой
Пафосный   ○○○○○ Приземлённый
Шутливый   ●●○○○ Серьёзный (находки и деньги — не мем)
```

### Примеры голоса

| Ситуация | ✅ Так | ❌ Не так |
|----------|--------|----------|
| Пустой каталог | «Пока тишина. Создай первый лот или загляни позже.» | «Нет данных для отображения.» |
| Ставка принята | «Ставка 4 200 ₽ принята. Вы лидируете.» | «Операция выполнена успешно!» |
| Ошибка оплаты | «Платёж не прошёл. Карта отклонена — попробуй другую или повтори через минуту.» | «Error 402 / Something went wrong» |
| Upgrade Pro | «Нужна статистика и чат по лоту — это в Pro.» | «Разблокируйте премиум-возможности экосистемы!» |
| Отзыв после сделки | «Сделка закрыта. Оставь отзыв — это помогает всем.» | «Пожалуйста, оцените транзакцию в личном кабинете.» |

### Язык интерфейса

- Локаль по умолчанию: **ru**.  
- Обращение: **ты**.  
- Деньги: `Intl` → `4 200 ₽`.  
- Время: относительно рядом с абсолютным («осталось 2 ч 14 мин» + дата окончания).  
- Жаргон платформы допустим, если он уже в продукте: *лот, ставка, карма, резерв*. Не выдумывать маркетинг-неологизмы.

## 3. Визуальное направление

### Название стиля

**«Морские чернила и патина»**

Метафора: вечерний берег — тёмная вода, известняк скал, окись меди на находке, песок под лампой. Не курортный постер и не музейная витрина.

### Принципы композиции

1. **Одна композиция на первый экран** — бренд/продукт как герой, не «дашборд виджетов».  
2. **Вещь важнее хрома** — фото лота доминирует; UI — рама, не конкурент.  
3. **Воздух с характером** — фон не плоский белый: лёгкий градиент тумана, зерно или мягкий свет.  
4. **Движение со смыслом** — таймер, появление ставки, смена лидера; не декоративный шум.  
5. **Карточки только для действия** — если рамка/тень не помогают понять или кликнуть, их нет.  
6. **Mobile-first** — палец, один главный CTA, читаемые цифры ставки.

### Анти-референсы (намеренно избегаем)

- Фиолетовые/неоновыe SaaS-градиенты на белом  
- Тёплый крем + терракота + «газетный» серпантинный антиквариат  
- Стеклянный glassmorphism ради тренда  
- Коллажи из плашек, бейджей и стат-полосок в hero

### Настроение в 4 словах

**Тёмная вода · окись · известняк · живая ставка**

## 4. Логотип (спека до отрисовки)

> Файл логотипа пока отсутствует. Ниже — бриф для дизайна; после появления ассетов раздел обновить.

### Идея знака

Монолитная **словесная марка** `Tavrida Lot` + компактный **монознак** для аватара/favicon.

Референс формы монознака (не финальный арт):

- силуэт **монеты / круга лота** с вырезом, напоминающим береговую линию или волну;  
- либо стилизованная **метка/бирка** с небольшим «пробой» патины;  
- избегаем: герба Крыма, якоря «как у всех», лупы, молотка аукциониста в лоб.

### Словесная марка

| Правило | Значение |
|---------|----------|
| Написание | **Tavrida Lot** (латиница в лого; в текстах то же) |
| Нельзя | TAVRIDALOT слитно в лого · TavridaLot camelCase · «Таврида Лот» в логотипе (в UI-копи — можно уточнить позже) |
| Подпись | без tagline в основном лого; tagline отдельно: «находки · торги · сообщество» |

### Технические требования (после отрисовки)

| Параметр | Значение |
|----------|----------|
| Охранное поле | ≥ высота буквы **T** со всех сторон |
| Мин. размер digital | слово 96 px ширина · монознак 24×24 |
| Мин. размер print | слово 25 mm · монознак 8 mm |
| Фон | на `ink`, `chalk`, фото с затемнением; не на пёстрых коллажах |
| Версии | full color · mono ink · mono chalk · mono patina |

### Запреты

- Не искажать пропорции, не обводить градиентом «радугой»  
- Не ставить в круг-аватар чужого цвета вне палитры  
- Не использовать старые эскизы после утверждения v1

## 5. Цветовая система

Палитра названа словами среды, не «Primary-500».

### Базовые роли

| Токен | HEX | Роль |
|-------|-----|------|
| `ink` | `#0B1F24` | Текст, шапка, тёмные поверхности |
| `ink-soft` | `#16333A` | Secondary surfaces, панели |
| `chalk` | `#F2F4F3` | Основной фон (холодный известняк, не крем) |
| `mist` | `#E4EAE8` | Вторичный фон, разделители |
| `foam` | `#FAFBFA` | Приподнятые поверхности |
| `patina` | `#1F7A6E` | Primary brand / ссылки / фокус |
| `patina-deep` | `#155E55` | Hover primary |
| `copper` | `#C47A3A` | Акцент ставки, «горячее», CTA вторичный |
| `copper-deep` | `#A35F28` | Hover copper |
| `sand` | `#C9B89A` | Нейтральный тёплый акцент, теги, скины |
| `muted` | `#5C6B70` | Вторичный текст |
| `line` | `#CDD6D4` | Бордеры |

### Семантика

| Токен | HEX | Использование |
|-------|-----|----------------|
| `live` | `#D94A2A` | Идёт аукцион, «вы перебиты», критичный таймер <5 мин |
| `success` | `#2F7D4A` | Ставка принята, сделка ОК |
| `warning` | `#C4922A` | Лимиты тарифа, мягкое предупреждение |
| `danger` | `#B42318` | Ошибки форм, блокировки |
| `info` | `#2B6B8C` | Нейтральные системные подсказки |

### Поверхности и тема

**v0.1 = light-first.** Тёмная тема — те же роли, инвертированные поверхности (`ink` становится фоном, `chalk`/`foam` — текстом), без отдельной «ночной» радуги.

### Контраст (WCAG)

| Пара | Минимум | Заметка |
|------|---------|---------|
| `ink` на `chalk`/`foam` | AAA для текста | основной body |
| `patina` на `chalk` | AA для UI-текста ≥16px | ссылки |
| Белый текст на `patina` / `ink` / `copper` | AA для кнопок | проверять крупный и мелкий |
| `muted` на `chalk` | AA для secondary | не использовать для критичных статусов |

**Правило:** смысловой цвет (live/success/danger) **не единственный** носитель смысла — дублируем иконкой или текстом.

### Градиенты (допустимы sparingly)

- Hero / пустые состояния: `ink` → `ink-soft` под углом 165°, opacity маска.  
- Патина: тонкий вертикальный `patina` → `patina-deep` только на primary button.  
- Запрет: мультиколор-меши, «aurora», фиолетово-розовые blob’ы.

## 6. Типографика

### Пара шрифтов

| Роль | Шрифт | Почему |
|------|-------|--------|
| **Display / бренд** | [**Unbounded**](https://fonts.google.com/specimen/Unbounded) | Геометрия с характером, хорошая кириллица, не «дефолтный Inter» |
| **UI / текст** | [**Onest**](https://fonts.google.com/specimen/Onest) | Читаемый гротеск для плотных интерфейсов и кириллицы |
| **Таблицы / моно** | **JetBrains Mono** или system `ui-monospace` | Цены в компактных колонках, ID лотов (опционально) |

Fallback: `Unbounded, "Segoe UI", system-ui, sans-serif` · `Onest, "Segoe UI", system-ui, sans-serif`.

### Шкала (mobile-first, rem)

| Токен | Size / Line | Использование |
|-------|-------------|----------------|
| `display` | 40/44 · md 56/60 | Hero, имя бренда на промо |
| `h1` | 32/38 · md 40/46 | Заголовок страницы |
| `h2` | 24/30 | Секции |
| `h3` | 20/26 | Карточка лота, модалки |
| `body` | 16/24 | Основной текст |
| `body-sm` | 14/20 | Вторичный, фильтры |
| `caption` | 12/16 | Мета, подписи к фото |
| `price` | 20/24 semibold · md 28/32 | Текущая ставка |
| `timer` | 16/20 tabular-nums medium | Таймер аукциона |

### Правила набора

- Заголовки: `font-display`, tracking чуть уплотнённый на display (`-0.em`).  
- Body: `font-sans` (Onest), tracking нормальный.  
- Цены и таймеры: `font-variant-numeric: tabular-nums`.  
- Макс. ширина текста: ~65 ch на статьях/правилах; на лоте описание может быть уже из-за колонки фото.  
- Не ставить display-шрифт на длинные абзацы.

## 7. UI-основа (токены интерфейса)

### Сетка и отступы

| Токен | px | Применение |
|-------|-----|------------|
| `space-1` … `space-12` | 4-базовая шкала | Tailwind-совместимо: 4, 8, 12, 16, 24, 32, 48… |
| Контент max | 1200 px | Каталог, форум |
| Лот (desktop) | 2 кол.: медиа ~58% / инфо ~42% | Фото важнее сайдбара ставки |
| Gutter mobile | 16 | — |
| Gutter desktop | 24–32 | — |

### Радиусы

| Токен | px | Где |
|-------|-----|-----|
| `radius-sm` | 6 | Чипы, инпуты |
| `radius-md` | 10 | Кнопки, мелкие панели |
| `radius-lg` | 16 | Модалки, крупные блоки |
| `radius-full` | 9999 | Аватары, FAB |

Фото лота: **лёгкий radius-md**, не «супер-скруглённые карточки Instagram».

### Тени

Меньше теней — больше границ `line` и сдвига поверхности `foam`/`mist`.

| Токен | Значение | Где |
|-------|----------|-----|
| `shadow-sm` | `0 1px 2px ink/6%` | Инпуты focus-adjacent |
| `shadow-md` | `0 8px 24px ink/10%` | Dropdown, popover |
| `shadow-lg` | `0 16px 40px ink/14%` | Модалки |

Не использовать цветные неоновые glow.

### Иконографика

- Стиль: **outline 1.75–2px**, скруглённые торцы, оптический баланс под 20/24 px.  
- Набор: Lucide или аналог с единым stroke — не мешать filled и outline в одном ряду.  
- Категории находок: простые пиктограммы (монета, кераамика, металл…) без реалистичных иллюстраций в навигации.  
- Цвет иконки = цвет текста рядом, кроме статусных.

### Motion

| Жест | Длительность | Easing |
|------|--------------|--------|
| Hover / press | 120–160 ms | `ease-out` |
| Появление ставки | 220 ms | spring лёгкий / `ease-out` |
| Модалка | 200 ms | fade + 4px translateY |
| Таймер critical | пульс opacity 1↔0.55, 1.2 s | только при <5 мин |

`prefers-reduced-motion: reduce` — отключаем пульс и параллакс, оставляем мгновенную смену состояния.

## 8. Компонентные принципы (не полный UI-kit)

Полный kit — следующий этап. Здесь — правила, чтобы `@tavrida/ui` не расползся.

### Кнопки

| Вариант | Вид | Когда |
|---------|-----|-------|
| **Primary** | `patina` fill, белый текст | Главное действие экрана (одна на view) |
| **Bid** | `copper` fill | Сделать ставку / перебить |
| **Secondary** | ghost / outline `ink` | Отмена, вторичное |
| **Danger** | outline/soft `danger` | Снять лот, удалить |
| **Ghost** | текст + иконка | Третичные |

Размеры: `sm` 32 · `md` 40 · `lg` 48. На mobile основной CTA лота — sticky bottom, не прятать в overflow.

### Поля ввода

- Фон `foam`, бордер `line`, focus ring `patina` 2px.  
- Ошибка: бордер `danger` + текст под полем, не только цвет.  
- Цена: суффикс `₽`, tabular-nums.

### Бейджи / статусы аукциона

| Статус | Цвет | Подпись |
|--------|------|---------|
| Черновик | `muted` | Черновик |
| Скоро | `info` | Начнётся … |
| Идёт | `live` | Идёт · … |
| Вы лидируете | `success` | Вы лидируете |
| Вас перебили | `live` | Ставку перебили |
| Завершён | `muted` | Завершён |
| Не состоялся (резерв) | `warning` | Резерв не достигнут |

### Карточки лота в каталоге

Не «карточка ради карточки»:

- фото edge-to-edge внутри ячейки сетки;  
- цена и таймер — самый крупный текстовый блок после названия;  
- рейтинг продавца — одной строкой, без отдельной «стат-полоски».

### Модалки / тосты

- Модалка: max-width 480 / 640, без лишних иконок-героев.  
- Toast: снизу на mobile, сверху-справа на desktop; success не зелёный «праздник», спокойный `success` + короткий текст.

## 9. Фото, иллюстрации, инфографика

### Фото лотов (контент пользователя)

| Правило | Да | Нет |
|---------|----|-----|
| Свет | ровный, видна фактура | жёсткий флэш «вспышка в упор» как норма бренда |
| Фон съёмки | нейтральный / ткань / стол | стоковый пляж с бокалами |
| UI поверх фото | градиент scrim снизу для текста | плавающие бейджи по центру героя |

Платформа **поощряет** несколько ракурсов и масштаб (монета/линейка) — это и UX доверия, и бренд «уважения к вещи».

### Брендовые иллюстрации

- Плоские или мягко объёмные, палитра бренда.  
- Сюжеты: находка в руке, стол разбора, берег без открытки, форумный разговор.  
- Без клип-арт «аукционный молоток + доллар».

### Empty states

Одна иллюстрация + заголовок + одно действие. Без трёх колонок «онбординг-фич».

### D3 / инфографика

- Серия цветов: `patina`, `copper`, `ink-soft`, `sand`, `info`.  
- Оси и подписи — `muted` / `body-sm`.  
- Легенда вне плота, не поверх.  
- Подробности — отдельный пункт в UX TODO.

## 10. Контент-паттерны

### Empty / Zero

Структура: **факт → что можно сделать → CTA**.  
«Нет избранного. Сохраняй лоты сердцевиной — они появятся здесь.» + кнопка в каталог.

### Ошибки

Человек + следующий шаг. Коды API наружу не светим (только в tech details / support).

### Уведомления (Novu)

Заголовок ≤ 60 символов, тело — одно предложение. Тон живой, без «Уважаемый пользователь».

### Email / соцсети

Те же цвета и Unbounded/Onest. Обложки: тёмный `ink` + одно фото находки full-bleed + словомарка. Без сетки из четырёх квадратов с акциями.

## 11. Примеры применения (целевые)

| Поверхность | Акцент |
|-------------|--------|
| Главная | Full-bleed атмосфера / герой с брендом, один CTA «Смотреть лоты», без стат-стрипа |
| Каталог | Сетка лотов, фильтры спокойные, `patina` на активных чипах |
| Страница лота | Фото + `copper` ставка + таймер; обсуждение ниже |
| Форум | Больше текста Onest, меньше декора; карма — спокойный бейдж |
| Wallet / тарифы | Ясная таблица Free/Basic/Pro, без «premium glitter» |
| Документация | Этот брендбук + `chalk` фон docs не обязан совпадать 1:1, но названия токенов — да |

---

## 12. Нельзя (антипримеры)

1. Два primary CTA на одном экране.  
2. Hero с четырьмя карточками преимуществ и счётчиками.  
3. Фиолетовый или радужный градиент «для современности».  
4. Inter/Roboto как основной UI-шрифт.  
5. Терракотово-кремовый «антикварный журнал» как дефолт.  
6. Цвет как единственный индикатор статуса.  
7. Анимация ради анимации на каждом ховере.  
8. Логотип на шумном фото без затемнения.  
9. Канцелярит и англицизмы без нужды («запустите experience торгов»).  
10. Скрывать цену и таймер ради «воздуха».
