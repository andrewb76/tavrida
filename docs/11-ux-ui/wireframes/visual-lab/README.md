## 🎯 Эталонный экран

**W03** `/auctions/:id` — галерея, мета, LIVE/таймер/цена, табы, sticky CTA.

Один контент (лот «Серебряный denarius · III в.»), три направления × **light / dark** (брендбук: light-first, dark = те же роли на инвертированных поверхностях).

---

## 🧭 Три направления

| ID | Название | Тезис | Риск |
|----|----------|-------|------|
| **A** | **Известняк + патина** | Light-first как в брендбуке: `chalk` фон, `patina` primary, `copper` на ставке | Может казаться «слишком спокойным» без атмосферы |
| **B** | **Тёмная вода** | Ink как сцена: тёмный shell, фото edge-to-edge, светящийся ask | Сложнее формы/админка; нужен light twin |
| **C** | **Редакционный каталог** | Типографика-герой, песочные акценты, меньше «аукционного жара» | Ближе к анти-референсу «музей» — держать дистанцию |

Все три опираются на палитру брендбука (`ink` / `chalk` / `patina` / `copper` / `sand`) и шрифты **Unbounded + Onest** — меняется **композиция и тон**, не набор HEX наугад.

## 🖥️ Как смотреть

1. Открыть [W03-visual-directions.html](./W03-visual-directions.html) → Live Preview / браузер.  
2. Переключать вкладки **A / B / C**.  
3. Сверить light/dark twins в `previews/` (выше).  
4. Смотреть на телефоне (или DevTools mobile) — CTA и фото.  
5. Отметить выбор в таблице ниже → каскад в tokens + 1–2 пилотных экрана.
