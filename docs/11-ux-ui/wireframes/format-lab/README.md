## 🎯 Как пользоваться

1. Открыть [W03-lot-page-formats.md](./W03-lot-page-formats.md) на GitHub Pages или локально.
2. Просмотреть форматы **A–E** (ASCII, Mermaid, component tree, matrix, HTML).
3. **Format E:** открыть [W03-lot-page.html](./W03-lot-page.html) → **Live Preview** (расширение `ms-vscode.live-server`, см. `.vscode/extensions.json`).
4. Отметить выбор в таблице «Ваш выбор» в конце W03 doc.
5. Обновить [wireframes/README.md](../README.md) — зафиксировать стандарт.
