## 🔗 Нормативные ориентиры (для подготовки L-20…L-24)

Список для **исследования юристом** (не исчерпывающий):

| Область | Акты / темы (РФ) |
|---------|------------------|
| Персональные данные | 152-ФЗ, подзаконные акты Роскомнадзора |
| Информация в сети | 149-ФЗ |
| Культурные ценности | Закон о культурных ценностях, вывоз, учёт |
| Охрана объектов культурного наследия | 73-ФЗ (публикация мест находок) |
| Археология | Закон об объектах археологического наследия |
| Административные штрафы | КоАП (статьи по охране памятников — уточнить с юристом) |
| Потребители | ЗоЗПП (если платформа берёт плату за услуги) |
| Электронная коммерция | 54-ФЗ (чеки), оферта |

Регион **Крым** — проверить актуальность применимости и региональные регламенты отдельно.
