# 🔔 Сервис: auction_subscriptions

> **Статус:** in progress · **Версия:** 0.1

## 🎯 Назначение

Подписки пользователей на аукционы и категории Tavrida Lot.

- Уведомления о новых лотах в интересующих категориях
- Отслеживание изменений ставок в избранных аукционах
- Проверка лимитов через `financial-policy`

## 📖 Термины

| Термин | Описание |
|--------|----------|
| **Subscription** | Подписка пользователя на категорию или конкретный аукцион |
| **Digest** | Сводное уведомление (email/push) |

## 🔗 Взаимодействие

| Сервис | Взаимодействие | Протокол |
|--------|----------------|----------|
| financial-policy | `POST /limits/check` | HTTP |
| auction | События создания/завершения | RabbitMQ |
| notifications | email / push | HTTP |
