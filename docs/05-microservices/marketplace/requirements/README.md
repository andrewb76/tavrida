# 🛒 Маркет услуг — требования

> **Статус:** draft · **Версия:** 0.1  
> **Спецификация сервиса:** [../README.md](../README.md)

## 🎯 Кратко

| Аспект | Решение |
|--------|---------|
| Модель | **Справочник услуг** — пользователи сами создают услуги и цены |
| Free тариф | **0** активных услуг — только заказчик, не поставщик |
| Портфолио | Публичное, per listing |
| Feedback | Как у аукциона → rating, карма |
| Expert | Роль от admin → экспертиза на **лоте** аукциона |
| Комиссия платформы | **Не определена** — отдельный юридический документ |

## 📋 Категории услуг (примеры)

- Реставрация и чистка
- Экспертиза и атрибуция
- Фотосъёмка для каталога
- Упаковка и доставка
- Хранение
